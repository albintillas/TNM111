# TNM111 Assignment 4 — Star Wars Network Comparison

## How to run

- Option A: Open this folder in VS Code and install the extension "Live Server". 

Then start the live server by pressing "Go Live" at the bottom right.

- Option B: Run a local static server with python from the project root:

"python -m http.server 8000" in the console.

Then open `http://localhost:8000`.
